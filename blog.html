What is the difference between map(), filter(), forEach() and find() ?

In JavaScript map() returns a new array with updated elements from calling a function for every single array elements. It does not change original array and it doesn't execute for empty array.

The filter() method creates a new array for all the elements of the array which satisfy certain conditions.

The forEach() method calls a function and iterates over the elements of an array.The forEach() method is used we dont need anything in return.

The find() method executes a function for each array element and returns only the value of the first element that satisfy certain conditions. The find() method returns undefined if no elements are found.
